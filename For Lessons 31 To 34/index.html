التكليف 04

دا كود استخدمته ف التكليف وحابه اطبق عليه واشرحلكم دا استخدمته ف اي 

                
                    window.onload = function() {
                        var dateInput = document.getElementById('date');
                        var monthInput = document.getElementById('month');
                        
                        // تعيين القيم الافتراضية
                        dateInput.value = '1982-10-25';
                        monthInput.value = '1982-10';
                      }
                
             
                1. `window.onload = function() { ... }`: هذا يعني أن الكود JavaScript سيتم تنفيذه عندما ينتهي تحميل الصفحة بالكامل. هذا يضمن أن العناصر المستهدفة في الكود موجودة في الصفحة قبل تنفيذ أي عملية عليها.
    
                2. `var dateInput = document.getElementById('date');`: هنا نستخدم `document.getElementById` للعثور على عنصر HTML الذي يحمل الـ id "date". هذا العنصر هو حقل الإدخال (input field) للتاريخ.
                
                3. `var monthInput = document.getElementById('month');`: هنا نستخدم `document.getElementById` مرة أخرى للعثور على عنصر HTML الذي يحمل الـ id "month". هذا العنصر هو حقل الإدخال (input field) للشهور.
                
                4. `dateInput.value = '1982-10-25';`: هذا السطر يعيّن القيمة الافتراضية لحقل التاريخ. نستخدم خاصية `value` لتعيين القيمة المحددة، وفي هذه الحالة هي "1982-10-25".
                
                5. `monthInput.value = '1982-10';`: هذا السطر يعيّن القيمة الافتراضية لحقل الشهور. نستخدم خاصية `value` لتعيين القيمة المحددة، وفي هذه الحالة هي "1982-10".
                
                باستخدام هذا الكود، ستتم تعيين القيم الافتراضية لحقول التاريخ والشهور عند تحميل الصفحة. يتم ذلك باستهداف العناصر المحددة باستخدام `document.getElementById` وتعيين القيم المطلوبة باستخدام خاصية `value`.
